image Media Fetching: YouTube, Images, GIFs

AwesomeBot provides useful commands for Google Image Search, Giphy access, and full YouTube search. In addition, you can fetch data from Wikipedia, Wolfram|Alpha, MSN Weather, and even links to apps on the Play Store and App Store.

refresh RSS Articles and Reddit posts

Reddit and RSS can become integral to the Discord experience with AwesomeBot. We allow server mods to set custom RSS feeds, and our Reddit viewer includes data beyond just the link. Fetching is fast and simple with our streamlined command processing.

person Discord User Profiles

Often, you want everyone to know something about you. With AwesomeBot, you can add information to your user profile across Discord to express yourself. Even if you don't use this, the capable profile command shows information about each user's account and activity on the server.

poll Full-Featured Per-Server Stats

One of the best features of AwesomeBot is our full-featured, per-server stats system that logs every message sent and game played. You can view up-to-date information about the most popular members and games. This even includes viewing messages you've been mentioned in over the past week and having AwesomeBot PM you if you're mentioned and you're offline.

play_circle_filled In-Chat Polls and Trivia Game

Have you ever wanted to easily ask the server a question? AwesomeBot provides live, in-chat polls that can be rapidly created and tallied. In addition, there's a fun group trivia game that even allows for custom question sets per-server.

card_giftcard AwesomePoints: Karma for Discord

Sometimes, you just want to show your agreement without having to send a long message. AwesomePoints allow you to quickly upvote users, accepting a wide range of syntaxes. This system is integrated throughout the bot: you'll lose points for breaking the rules, get rewards for being the most active on the server, and you can even buy lottery tickets!

settings Configurable Per-Server with Mod Tools

Don't like a feature in AwesomeBot? Disable it. Our powerful admin console allows server mods to control every aspect of the bot per-server, as well as gain access to our comprehensive mod tools. This includes a spam filter, NSFW content filter, blocklist, and strikes system to keep track of when users break the rules.

extension Easily Extensible with JavaScript

The possibilities with AwesomeBot are endless. Maybe you want to add a live score tracker. Or a Spotify search command. Our robust extension system allows you to write a little JavaScript code to enrich the bot's functionality without worrying about breaking it.
